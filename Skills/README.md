# Skills

This folder contains the active Copilot CLI skills for the quality analytics system.

Each skill is written as a delegated analyst assignment. It should tell Copilot CLI how to inspect data, perform analysis, evaluate business impact, prioritize findings, and produce a decision-ready recommendation.

## Active Skills

- `claims-analysis`
- `margin-impact-analysis`
- `order-book-analysis`

## Skill Standard

Each skill should include:

1. Purpose
2. When to use
3. Business questions answered
4. Required inputs
5. Optional inputs
6. Analysis steps
7. Metrics and calculations
8. Prioritization logic
9. Output format
10. Recommended visuals
11. Guardrails
12. Example prompts
13. Definition of done
